# E-commerce SQL Analytics Project

## Overview
This project focuses on SQL-based business analysis using an E-commerce retail dataset.

The project includes:
- KPI analysis
- Customer analytics
- Product performance analysis
- Shipping analysis
- Return rate analysis
- Cohort retention analysis

## Database Schema

The database contains 5 tables:

- customers
- orders
- order_items
- products
- shipping

## Skills Demonstrated

- SQL JOINs
- Aggregations
- Window Functions
- CTEs
- Cohort Analysis
- Retention Analysis
- Business KPI Analysis

## Business Questions Solved

- Monthly revenue trends
- Customer lifetime value
- Product profitability
- Shipping efficiency
- Return behavior
- Retention analysis

## Tools Used

- SQL Server
